一些使用事项:
	server:
	编辑server.cfg
		nickname:
			nickname没有使用,不需要编辑
		host:
			如果内网穿透一定用127.0.0.1 (ipv4)或者::1 (ipv6)
			如果服务器有公网ip, host请填写公网ip (包括ipv4和ipv6)
		port:
			默认端口45001,如果更改注意冲突(最好设置为45001)
		
		一些说明:
		选项前方的值(例如nickname)不要随意改变或者挪动位置,
		+后方要填写的内容
	
	client:
		host:
			填写ip即可
			
			
	值:
		nickname= ,设置昵称, server没有用
		host= ,设置ip地址
		port= ,设置端口
		IP= ,设置ip协议, 可以填入的值: ipv4, ipv6
		language= ,设置地区语言, 可以填入的值: en-us, zh-cn

	Some usage precautions:
	Server:
		Edit server.cfg
		Nickname:
			Nickname is not in use and does not require editing
		Host:
			If the intranet penetration must be 127.0.0.1 (ipv4) or: 1 (ipv6)
			If the server has a public IP address, please fill in the public IP address for the host (including ipv4 and ipv6)
		Port:
			The default port is 45001. If changing, please pay attention to conflicts (preferably set to 45001)

		Some explanations:
			Do not arbitrarily change or move the position of the value in front of the option (such as nickname),
			+The content to be filled in later
				Client:
					Host:
						Just fill in the IP address
		
		Value:
			nickname=, setting a nickname, server is not used
			host=, set IP address
			port=, set port
			IP=, set IP protocol, can fill in values: ipv4, ipv6
			language=, set the region language, and the values that can be filled in: en-us, zh-cn
